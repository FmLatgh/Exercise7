# Practice: Specialisation Web

If you already feel comfortable with object literals, you can skip this assignment!


## Learning goals
* You can use object literals to hold different variables and values that have a relationship

## ASSIGNMENT
* Open the folder `exercise_ppt_game` in Webstorm.
* Try to understand the code in the different files.
* Ask questions if there is something you don't understand.
* Open main.html in the browser. In the console you will see errors. 
* Try to fix the code by solving the TODO's (comments starting with `TODO:`)in the code.
* Use the debugger in the browser if necessary.
